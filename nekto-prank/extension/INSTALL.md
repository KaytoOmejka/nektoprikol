# 🎙 Голосовой мост nekto.me — расширение для Firefox

Связывает **два** голосовых чата nekto.me: два собеседника начинают слышать друг
друга, ты слышишь обоих и можешь в любой момент говорить так, чтобы тебя слышали
оба.

## Как это работает (коротко)

Сигналинг голосового чата nekto зашифрован, поэтому «притвориться клиентом» из
программы нельзя. Вместо этого расширение работает **внутри настоящих вкладок
nekto**: сам сайт ведёт звонки как обычно, а расширение перехватывает уже готовый
звук и перенаправляет его. Шифрование при этом не трогается.

```
Вкладка A (аккаунт A) ──┐  голос A → Б,  голос Б → A   ┌── Вкладка B (аккаунт B)
   Собеседник A         │  + твой микрофон обоим        │      Собеседник B
                        └── ты слышишь обоих ───────────┘
```

## Что понадобится

- **Firefox**.
- **Два аккаунта nekto** в **двух контейнерах** (см. шаг 1) — это обязательно: два
  собеседника = две независимые сессии.
- **Наушники** (без них микрофон поймает звук из колонок → эхо и свист).

---

## Шаг 1. Два аккаунта в одном окне — контейнеры

Две обычные вкладки nekto.me в одном Firefox делят один аккаунт. Чтобы было два
разных собеседника, нужны **контейнеры** (изолированные «личности» в одном окне):

1. Установи дополнение **Firefox Multi-Account Containers**:
   <https://addons.mozilla.org/firefox/addon/multi-account-containers/>
2. Оно добавит значок контейнеров на панель. Создай два контейнера, например
   «Чат 1» и «Чат 2».

> Почему именно контейнеры, а не два окна/профиля: расширению нужно, чтобы обе
> вкладки жили в **одном** Firefox — только тогда оно может связать их между собой.
> Контейнеры дают два разных аккаунта в одном окне.

## Шаг 2. Установить расширение

1. Открой в Firefox `about:debugging` → слева **This Firefox**.
2. **Load Temporary Add-on…** → выбери файл `extension/manifest.json` из этой папки.
3. Готово. (⚠️ Временное дополнение слетает при перезапуске Firefox — после
   перезапуска просто повтори этот шаг.)

## Шаг 3. Открыть два голосовых чата

1. Открой `nekto.me` в контейнере **«Чат 1»**, войди в первый аккаунт, зайди в
   **голосовой чат** и начни искать собеседника.
2. Открой `nekto.me` в контейнере **«Чат 2»** (вторая вкладка), войди во второй
   аккаунт, тоже зайди в голосовой чат и начни искать.
3. В правом нижнем углу каждой вкладки появится панель **🎙 Голосовой мост**.
   Когда обе вкладки нашли собеседников и связались, статус будет
   **«звонок активен ✓»**.

С этого момента:

- **Собеседники слышат друг друга** автоматически.
- **Ты слышишь обоих** (каждая вкладка играет своего собеседника — держи звук
  включённым в обеих).
- Кнопка **🎙 Микрофон** на панели — включает/выключает твой голос **сразу для
  обоих** собеседников. Нажми в любой из вкладок. Когда включён — под кнопкой
  бежит зелёная полоска уровня (видно, что голос идёт).

Про панель: её можно **перетаскивать** за шапку и **сворачивать** кнопкой «—»
(положение запоминается). Цветная точка слева = статус: жёлтая — подключается,
оранжевая — ждёт вторую вкладку, зелёная — всё работает. Подпись «вкладка 1/2»
помогает не путать окна.

## Если кто-то вышел / «искать заново»

Жми поиск нового собеседника прямо в nekto, как обычно. Мост сам переподцепится к
новому звонку — отдельных действий не нужно.

---

## Если что-то не так

- **Нет панели** — обнови вкладку nekto; проверь, что расширение загружено в
  `about:debugging`.
- **Статус застрял на «жду вторую вкладку»** — значит вторая вкладка nekto не
  открыта/не в голосовом чате, или открыта в том же контейнере (нужны разные).
- **Эхо/свист** — надень наушники (микрофон ловит колонки).
- **Тебя не слышат** — нажми 🎙 Микрофон (должно стать «ВКЛ»); при первом включении
  Firefox спросит доступ к микрофону — разреши.
- **Диагностика**: открой DevTools (F12) → Console в каждой вкладке. Сообщения с
  меткой `[мост]` показывают этапы: перехват звонка, связь между вкладками,
  подмена исходящего звука.

## Честные оговорки

- Это обход штатного поведения nekto. Если nekto обновит свой сайт, расширение
  может перестать работать — тогда его придётся чинить.
- Только для собственного контента и под твою ответственность.

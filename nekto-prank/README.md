# nekto.me · мост (для пранков)

Локальный сайт, который соединяет двух случайных собеседников из **nekto.me**
друг с другом через бота и пересылает их сообщения. Ты видишь весь диалог в
одном окне и можешь в любой момент написать **от лица любого из них**, чтобы
поймать реакцию.

```
Собеседник 1  ↔  бот(клиент 1)  ↔  [ТЫ]  ↔  бот(клиент 2)  ↔  Собеседник 2
```

> Только для твоего собственного контента. Используются твои токены nekto.me.

## Запуск

```bash
./start.sh
```

или двойной клик по **`start.command`** в Finder (если macOS ругается —
правый клик → «Открыть»).

Скрипт сам создаст виртуальное окружение, поставит зависимости и откроет
интерфейс на <http://127.0.0.1:8765>.

Зависимости: Python 3.10+, `aiohttp`, `python-socketio` (ставятся автоматически).

## Как достать 2 токена nekto.me

Нужны **два** токена — по одному на каждого собеседника (удобно открыть
nekto.me в двух окнах кнопками «Открыть nekto.me» прямо в приложении).

Самый простой способ встроен в приложение: на экране настройки раскрой
**«❓ Как получить токен»** и следуй шагам. Коротко:

**Способ 1 — кнопка-закладка (проще всего, настраивается один раз):**

1. Включи панель закладок браузера: `Ctrl`+`Shift`+`B` (Mac: `⌘`+`Shift`+`B`).
2. Перетащи мышкой кнопку **🔑 Взять токен nekto.me** из приложения на эту панель.
3. Открой nekto.me, начни искать собеседника и **немного попереписывайся**
   (5–10 коротких диалогов) — так аккаунт считается «живым» и бот ищет без капчи.
4. На странице nekto.me нажми закладку **🔑 Взять токен** — токен скопируется
   в буфер. Вставь его в поле «Токен» (`Ctrl`+`V`).
5. Повтори для второго собеседника.

**Способ 2 — через консоль** (если закладка не сработала): на nekto.me нажми
`F12` → вкладка **Console**, вставь и нажми Enter:

```js
copy(JSON.parse(localStorage.getItem("storage_v2")).user.authToken)
```

Токен скопируется в буфер — вставь в приложение.

Поле **User-Agent** заполняется автоматически — менять не нужно.

## Как пользоваться

1. Вставь оба токена → **Соединить**. Статусы пойдут: *подключение → ищет →
   общается*.
2. Реальные реплики обоих появляются в общем окне и автоматически пересылаются
   между собеседниками (тумблер **Автопересылка**).
3. Чтобы вмешаться — выбери **«Писать как: Собеседник 1/2»**, набери текст и
   отправь. Второй собеседник получит это как реплику первого. В твоём окне
   такое сообщение помечается жёлтым **«вставлено»** (видно только тебе).
4. **Новый поиск** — переподключить обоих к новым людям. **Стоп** — отключиться.

## Если что-то не работает

- **Не находит собеседника / сразу отключает** — токен «холодный». Попереписывайся
  на nekto.me ещё немного и возьми токен заново.
- **Совсем не подключается** — открой консоль браузера (F12): там в `console.debug`
  печатаются сырые события `[nekto raw]`. По ним видно, если nekto.me поменял
  протокол (например, поле с id пользователя или версию). Пришли их мне — поправлю.
